## 🚀 Portfolio

### 📅 勤怠管理システム
> Vue 3 + Node.js + Firebase で構築した勤怠・勤務報告管理Webアプリ

- 勤怠打刻・勤務報告の提出／承認フロー・ユーザー管理・CSV出力
- **AIチャット**（ChatGPT API）による使い方案内
- **RAG機能**（Gemini Embedding + Pinecone）による社内ナレッジ自律回答
- 設計〜実装〜デプロイまで個人で一貫して対応
